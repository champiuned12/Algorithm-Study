## 🤙 Rules!

### ⏰ 시간

**매주 목 19 : 30**<br>
**매주 일 19 : 00**

### 📝 진행

**→ 코드 리뷰를 위해 스터디 전 날까지 Github PR!**

**→ 이전 필수 문제 및 개인 문제 접근 방식 공유**

**→ 발표자 선정 후 다음 스터디까지 해당 주제 명확하게 인지 후 주제에 맞는 알고리즘 문제 선정**

**→ 알고리즘 문제 실전과 같이 30분간 동안 문제 풀이**

**→ 어려웠던 점 공유**
